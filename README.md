
# Laravel+Vuejs Waiton Japanese

## Tech Specification

- Laravel 8
- Vuejs
- Mysql

## Features
- Registration and Login
- Screen Recording
- Dashboard

## Installation

```bash
$ git clone https://github.com/jenry23/waiton-dev.git
$ cd waiton-dev/
$ composer install
$ cp .env.example .env
update .env and set your database credentials
$ php artisan key:generate
$ php artisan migrate
$ php artisan db:seed
$ npm install
$ npm run dev
$ php artisan serve

```
